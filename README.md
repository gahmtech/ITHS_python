# ITHS_python
API Catch the Flag

Ett kort program för att interagera med ett test-API för att fånga
en flagga.

Varför inte manuellt?
För att utgångstiden på erhållen TOKEN är för kort för att hinna skriva allt manuellt.

Detta program erhåller först en TOKEN för att sedan verifiera den och erhålla en SECRET-key
som sedan snabbt används för att erhålla det ultimata slutet: EN FLAGGA!

En del av programmeringskursen för att konsumera API.
